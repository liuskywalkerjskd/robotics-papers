# 机器人论文集

使用 Claude Code 生成的 arxiv 机器人领域论文总结。

## 简介

本仓库收集了 arxiv 上机器人领域的论文总结，由 Claude Code 的论文阅读技能生成。每个文件夹包含：
- 原版论文（PDF/HTML）
- 提取的文本
- 关键要点总结
- 重要图表

## 使用方法

每个论文文件夹以 arxiv ID 命名。进入文件夹即可阅读总结。

## 贡献指南

添加新论文时：
1. 创建以 arxiv ID 命名的文件夹（如 `2603.12345`）
2. 添加论文总结（README.md）、PDF 和图表
3. 更新本 README 和英文版 README
