# Robotics Papers

[中文版](./README_zh.md) | English

Collection of arxiv robotics papers with Claude Code summaries.

## About

This repository contains summaries of robotics papers from arxiv, generated using Claude Code's paper-reading skill. Each folder contains:
- Original paper (PDF/HTML)
- Extracted text
- Summary with key insights
- Important figures

## Usage

Each paper folder is named with its arxiv ID. Navigate to the folder to read the summary.

## Contributing

When adding a new paper:
1. Create a folder named with the arxiv ID (e.g., `2603.12345`)
2. Add the paper summary (README.md), PDF, and figures
3. Update this README with the new paper link
